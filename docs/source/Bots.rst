Bots
=========================================
In oTree, it is possible to deploy bots for conducting studies. Additionally, it is feasible to assess server load, for instance on platforms like Heroku, by utilizing an Amazon Web Server (AWS).
The functionality of the bots is aligned with developers' expectations regarding participant behavior.
In this context, we are developing a bot that inputs randomly generated values
