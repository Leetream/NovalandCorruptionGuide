=========================================
Deployment
=========================================
In this chapter, we will discuss how to upload an oTree project to Heroku, a cloud platform that allows for easy deployment of web applications.
We will provide an overview of Heroku, including its features and benefits for developers.

We will also cover the individual settings that can be customized when deploying an oTree project to Heroku, including environment variables, buildpacks, and add-ons.
These settings can help to optimize the performance and functionality of the application.
